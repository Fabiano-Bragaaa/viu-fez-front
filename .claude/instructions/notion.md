# Notion Documentation

When the user asks to document something in Notion, always create a dedicated page for the subject inside the **viu-fez-front** page (ID: `341f7aa3-c39d-8179-b80b-d45f36f509f5`).

- Explain the full flow in detail
- Include useful links to official documentation for each library or resource used
- Explain how the parts connect to each other, not just describe them in isolation
- Never add icons to pages
- Never add a generic description to the parent page
